# Модель клієнт-сервер

Модель клієнт–сервер - це спосіб організації роботи програм, де:
- **Клієнт** - це програма або пристрій, який робить запити на отримання послуг або ресурсів.
- **Сервер** - це програма або пристрій, який надає ці послуги або ресурси.
